## 📂 Folder Structure

Each workshop folder contains:

```text

├── README.md           # overview of the workshop
├── slides.pdf          # presentation slides
├── code/               # Code examples (if there was any)
│   └── solutions/      # solutions for exercises
└── exercises/          # Practice problems
```
